# Ecommerce Product Service

## Description
Manages product catalog including products, categories, and brands.

## Tech Stack
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Docker

## Responsibilities
- Create / update products
- Manage categories
- Product search & filtering

## APIs
- GET /products
- POST /products
- GET /products/{id}

## Database
- products
- categories
- brands
